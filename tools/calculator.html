How to Use this Calculator

This calculator allows you to perform both basic and advanced mathematical calculations. Use the buttons to input numbers and operations. You can switch between normal and scientific modes using the "Scientific Mode" toggle button. In scientific mode, you have access to functions like sine, cosine, tangent, logarithms, square root, and more.
